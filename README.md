Sistema de Gestão para Centro de Treinamento (Streamlit)

Este projeto surgiu a partir de um problema real: um centro de treinamento que realizava todo o controle de alunos, presenças e finanças de forma manual, utilizando folhas de papel.

A principal dor era clara: risco de perda de dados, falta de organização e dificuldade para análise das informações.

A solução foi o desenvolvimento de uma aplicação interativa utilizando Python + Streamlit, transformando um processo manual em um sistema digital simples, eficiente e acessível.

🚀 Objetivo

Criar uma aplicação que permita:

Centralizar e organizar dados dos alunos
Acompanhar pagamentos e pendências
Monitorar presença e pontuação
Gerar insights rápidos para tomada de decisão
🛠️ Tecnologias Utilizadas
Python
Streamlit
Pandas

O uso do Streamlit permite transformar scripts Python em aplicações web interativas sem necessidade de front-end, acelerando o desenvolvimento e facilitando o uso por qualquer pessoa.

📊 Funcionalidades

A aplicação oferece:

🔍 Busca e Filtros
Pesquisa de alunos por nome
Filtro de clientes com pagamentos pendentes
📈 Indicadores Gerais
Total de alunos
Quantidade de alunos pagantes
Pagamentos pendentes
Total de pontos acumulados
🏆 Ranking de Alunos
Visualização dos alunos com maior pontuação
Destaque para o Top 5
📁 Exportação de Dados
Download dos dados filtrados em CSV
💡 Problema Resolvido

Antes:

Controle manual em papel
Alto risco de perda de informações
Difícil análise de dados

Depois:

Dados organizados e centralizados
Redução de retrabalho
Acesso rápido a informações relevantes
Maior controle operacional
🎯 Diferencial do Projeto

Diferente de ferramentas tradicionais de BI, este projeto foca em:

Simplicidade de uso
Atualização dinâmica via CSV
Interatividade imediata
Facilidade de manutenção

Isso torna a solução ideal para pequenos e médios negócios que precisam de organização sem complexidade.
